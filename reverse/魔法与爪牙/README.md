想想其他的办法。在之前的几篇文章中，我介绍了怎样通过wireshare或者fiddler来抓取软件和后台服务器之间的协议。但是这次游戏和后台服务器之间的通信是https链接，无法直接看到明文的协议内容。对于这种情况我们有以下几种应对措施
1，逆向工程整个软件并且通过分析源代码来弄明白协议。
2，用API注入技术拦截ssl_write()和ssl_read()这两个函数。
3，利用中间人来代理ssl链接。
本文主要讲述第三种方法。
